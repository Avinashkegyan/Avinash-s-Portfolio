## Overview

In this project, I developed a flexible and scalable code system using HTML, CSS, SQL, PHP for back-end development. The goal was to create a reusable component library that not only adheres to consistent design principles but is also easily extendable for future needs. The design system was aimed at improving the overall developer experience while maintaining visual consistency across multiple projects.

# Hospital Management System

## Description
The Hospital Management System is a comprehensive web-based application designed to improve the management of patient records, appointments, and medical prescriptions. The system enables patients to register, log in, and view their medical records online. Doctors can access patient information, schedule appointments, prescribe medications, and manage various aspects of healthcare via a robust admin panel. The project focuses on enhancing the efficiency, security, and care provided by hospitals. Additionally, the system supports features like patient information management, appointment tracking, and an admin management module for hospital staff.

## Features
- Patient registration and login.
- Secure access to patient records.
- Appointment scheduling and tracking.
- Admin management module for hospital staff.
- Prescription management.
- Enhanced care and efficiency.

## Technologies
- HTML, PHP, MySQL
